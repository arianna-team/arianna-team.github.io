New Features for Partners and EU Vulnerability Monitoring

Edition: March 2026

Product Update

This edition highlights two platform updates that strengthen partner operations and improve vulnerability monitoring coverage.

1. New authorization layer for partners

ARIANNA now includes a new authorization layer that allows partners to manage their own customers directly within the platform. This makes it easier to support multi-customer operating models while maintaining clear separation, access control, and operational accountability.

  • Partners can manage customer environments with clearer ownership boundaries.
  • Access can be aligned with partner service responsibilities.
  • The model supports more scalable onboarding and day-to-day customer operations.

2. Connection to the EU CVE vulnerability database

ARIANNA is now connected to the EU CVE vulnerability database, expanding the platform's visibility into relevant vulnerability intelligence sources. This helps teams monitor emerging issues more effectively and strengthens coverage for product security and compliance workflows.

  • Broader vulnerability monitoring across relevant public sources.
  • Improved support for regulatory and regional security expectations.
  • Better visibility for teams managing connected and embedded products in Europe.

Together, these updates improve both platform scalability for partners and visibility for vulnerability management teams working in regulated product environments.
